# Logo Assets

Please save your Landee logo images in this directory with the following names:

1. **favicon.png** - Icon only (shield/house logo) for browser favicon
   - Recommended size: 256x256px or 512x512px
   - Transparent background preferred

2. **logo-full.png** - Full logo with text "Landee Property Management System"
   - Recommended height: 80-120px
   - Transparent background preferred
   - Used in: Landing page header, Sidebar (expanded), Social media previews

## Where they're used:

- `favicon.png`: Browser tab icon, Login page, Register page, Collapsed sidebar
- `logo-full.png`: Landing page header, Expanded sidebar, Open Graph/Twitter Card images
